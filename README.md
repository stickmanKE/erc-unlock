# Toyota ERC Unlock Tool

A free, offline-capable web tool to generate unlock codes for Toyota Japanese head units after a battery disconnect.

🔗 **Live:** [erc-unlock.vercel.app](https://erc-unlock.vercel.app)

## Supported Models
All W5x–W6x series including:
- NSCD-W66
- NSCP-W64
- NSZT-W66T / W64 / W62G
- And many more

## Features
- ⚡ Instant code generation — no server, works offline
- 🌙 Light / Dark mode
- 📋 Copy to clipboard
- 💬 Share result via WhatsApp
- 🖨️ Print / Save as PDF
- 🕓 Lookup history (stored locally)

## How to Get Your ERC Code
1. Press and hold **[CURRENT LOCATION]** on the unit while toggling the small lamp ON/OFF 3–4 times
2. Navigate to **Security Code Entry** in the service menu
3. Note the 16-digit ERC code displayed
4. Enter it in this tool

## Algorithm
Based on the open-source reverse-engineering work by:
- [alexeyfadeev/erc-calculator](https://github.com/alexeyfadeev/erc-calculator) (C#)
- [Berks/python_erc](https://github.com/Berks/python_erc) (Python)
